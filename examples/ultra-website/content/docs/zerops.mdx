You can use this yml for importing a project via import wizard from the dashboard to deploy a [basic barebone project](https://github.com/zeropsio/recipe-ultra-deno).

```yaml
project:
  name: recipe-deno
  tags:
    - zerops-recipe

services:
  - hostname: app
    type: deno@1
    buildFromGit: https://github.com/zeropsio/recipe-ultra-deno
    enableSubdomainAccess: true
```

Use this `zerops.yml` in root of your project. This tells Zerops how to build and run your ultra project.

```yml
zerops:
  - setup: app
    build:
      base: deno@1
      buildCommands:
        - deno task build
      deployFiles:
        - .ultra/~
    run:
      base: deno@1
      ports:
        - port: 8000
          httpSupport: true
      start: deno task start
```

For more information vist [Zerops Docs](https://docs.zerops.io/) and [Zerops Discord Server](https://discord.gg/xxzmJSDKPT).
